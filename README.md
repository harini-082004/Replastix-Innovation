# Replastix-Innovation
Salesforce based  plastic waste management system
Project Title: Replastix Innovation
Overview:
Replastix Innovation is a smart and sustainable solution designed to manage plastic waste effectively. The project focuses on collecting, recycling, and converting plastic waste into useful products such as construction materials or reusable items.
This system aims to reduce environmental pollution and promote eco-friendly practices by using modern technologies and efficient waste management methods.
Features:
 Plastic Waste Management System
 Waste Tracking and Monitoring
 Recycling Process Management
 Real-time Data and Reports
 User-friendly Interface
 Automation of Waste Collection & Processing
 Eco-friendly Product Generation
Modules:
User Management Module
Handles admin and user details
Login and authentication
Waste Collection Module
Records plastic waste collection data
Tracks sources (households, industries)
Waste Segregation Module
Categorizes plastic types
Prepares waste for processing
Recycling Module
Manages recycling process
Converts waste into reusable materials
Product Management Module
Stores details of products created
Tracks usage and sales
Reporting Module
Generates reports and analytics
Shows environmental impact
